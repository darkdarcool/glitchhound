# GlitchHound

> A soon to be developed debugger library for Node

## Installation

```sh
yarn add glitchhound
```

## Usage

```js
import { Logger } from 'glitchhound'
let logger = new Logger('myDebug.log', true);
logger.log("Hello!");
```
